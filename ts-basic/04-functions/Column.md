# Column: Chapter 04 の補足

## 詰まりそうなポイント：関数オーバーロード
同じ名前の関数で、引数のパターンによって戻り値の型を変えたい場合があります。これを「オーバーロード」と呼びますが、実装側で全てのパターンを網羅する型を書く必要があり、少しコツがいります。

## 豆知識：アロー関数 vs Function宣言
TypeScript でも JavaScript と同様にアロー関数がよく使われます。
```typescript
const add = (a: number, b: number): number => a + b;
```
型注釈を「変数の方に書くか」「関数の括弧に書くか」選べるのが TS の面白いところです。
